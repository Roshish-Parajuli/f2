# Data Extraction 101

Data extraction is the process of retrieving data from various sources. This is the first step in the ETL (Extract, Transform, Load) process.

## Sources

Data can be extracted from a variety of sources, including:

*   Databases
*   APIs
*   Web pages
*   Files (CSV, JSON, XML, etc.)

## Techniques

There are several techniques for extracting data, including:

*   **Full Extraction:** Extracting all the data from the source.
*   **Incremental Extraction:** Extracting only the data that has changed since the last extraction.
*   **Web Scraping:** Extracting data from web pages.
